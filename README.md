# Portfolio

### 1_breakout_game
經典的打磚塊遊戲，移動滑鼠控制板子，隨著分數增加球速也越來越快，目標把磚塊消光光，拿到100分！  
1.	使用OOP架構遊戲和製作出圖形物件。  
2.	對具asynchronous特性的mouse listener設置開關閥，在點擊滑鼠啟動遊戲後，再次點擊不會再動作。  
3.	設置boundary條件，讓球碰到板子、磚塊和遊戲邊界時會反彈，並偵測球撞擊位置來消除方塊。  
Demo: https://youtu.be/S__GPsoL2EY
