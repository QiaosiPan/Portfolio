## 1_breakout_game
經典的打磚塊遊戲，分數增加球速也越來越快，目標把磚塊消光拿到100分！
1.	Asynchronous：對永遠偵測的mouse listener設置動作開關閥，點擊滑鼠開始遊戲到當局結束前，再次點擊不會對遊戲產生作用
2.	OOP：製作出圖形物件，並偵測球撞擊位置來消除方塊
3.	Boundary：確實設置boundary條件，讓板子可以跟著滑鼠移動，球碰到板子、磚塊和遊戲邊界時會反彈
